Configures git to autocorrect mistyped commands.

- Use `git config --global help.autocorrect 1` to enable git's autocorrect.

```shell
git config --global help.autocorrect 1
```

```shell
git config --global help.autocorrect 1
git status # Runs `git status` instead
```
